Keyboard Shortcuts

[CTRL] or [SHIFT]

If you desire to select multiple units that are scattered throughout the map, hold down either of these keys and select the units. The editor will automatically keep the other units selected so that you may modify them simultaneously.

[CTRL]+N

Creates a new map.

[CTRL]+O

Opens an existing map.

[CTRL]+S

Saves the current map.

[CTRL]+C

Compacts the current map.

[CTRL]+Z

Undo the last performed action.

[CTRL]+Y

Redo the last performed action.

[Delete]

Clear the selected object.

[CTRL]+A

Select all available objects.

[Enter]

Modify unit properties.

[CTRL]+G

Toggle the Tile Grid On/Off

[CTRL]+T

Switch to the Terrain layer.

[CTRL]+D

Switch to the Doodad layer.

[CTRL]+U

Switch to the Unit layer.

[CTRL]+L

Switch to the Location layer.

[CTRL]+F

Switch to the Fog of War layer.

[CTRL]+G

Go to the Start Location

[CTRL]+[Enter]

Modify the Player Settings

[CTRL]+[SHIFT]+F

Open the Forces dialog box.

[CTRL]+[SHIFT]+S

Open the Sound Manager

[CTRL]+[SHIFT]+T

Open the Triggers dialog box.

[CTRL]+[SHIFT]+B

Open the Mission Briefing dialog box.

[CTRL]+[SHIFT]+H

Open the Global Unit and Hero Settings dialog box.

[CTRL]+[SHIFT]+U

Open the Global Upgrade Settings dialog box.

[CTRL]+[SHIFT]+A

Open the Global Special Abilities dialog box.

[CTRL]+[SHIFT]+[Enter]

Open the Scenario Properties dialog box.

[CTRL]+B

Open a new Brush Palette tool menu.

[CTRL]+P

Open a new Doodad Palette tool menu.

[CTRL]+W

Create a New Map window.

[PgUp]

Moves the main map window up one screen for each press of the key.

[PgDown]

Moves the main map window down one screen for each press of the key.

[CTRL]+F4

Close the current window.

[CTRL]+F6

Switch to the next available window.

[ALT]+F4

Exit the Starcraft Campaign Editor.

[F1]

Open the Help Menu

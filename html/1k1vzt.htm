Step 1: Starting a New Map

From the File menu, select New. A New Scenario dialog will appear.

In the dialog box, under the Scenario Size section, you will see a pull-down menu for Width. Select 64 as your Width. Since the Square Map checkbox is enabled, the Height control automatically updates to match your selection for Width.

Next, under the Scenario Type section, select Space Platform as your tileset from the Tileset list. A tileset consists of a themed set of terrain types that can be used to make a map. The Space Platform tileset contains man-made metal platforms and starry space.

In the Default Terrain list, select Space. Default Terrain is the terrain type that will fill the map when it is first created.

Finally, click the OK button.

After a moment, you will see the main window appear. It will be completely black, since we have chosen Space as our Default Terrain. You can start creating space platforms in the next section.

Step 6: Saving and Playing the Map

From the File menu, select Save. A save dialog box will appear.

The editor defaults to saving maps in the Starcraft Maps directory. You must save your maps to this directory in order to load them from the game.

Make sure you are in the Starcraft Maps directory, type in a name for your map, and then press Save.

Your map is now ready to be played in Starcraft. Exit the Campaign Editor and run Starcraft.

Choose Play Custom from the Starcraft single player Campaign Screen. A load screen with a view of the Maps directory will appear.

Choose your map from the list. In the display just below the file list, you will see slots for two players. The first slot will be for you, the human player, and the second will be for the computer.

Click OK. After a moment, you will be in the game.

Notice that you start with a colony center and four workers, even though you never placed any units on your map. In all Game Types except “Use Map Settings”, Starcraft automatically creates four workers and a colony center at the start location for each player.

That concludes the tutorial. You now have the basic skills for creating maps and playing them in Starcraft.
